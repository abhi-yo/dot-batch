# Testimonial Slider Web App

## Introduction

This is a single-page web application built with React.js that showcases testimonials from various individuals. The application features a testimonial slider with navigation buttons for easy exploration.

## Features

- **Testimonial Slider**: Displays testimonials in a visually appealing slider format.
- **Navigation Buttons**: Allows users to move forward or backward in the slider to view different testimonials.
- **Surprise Button**: Presents a random testimonial for a delightful user experience.

## Technologies Used

- **React.js**: Used for building the user interface and managing the application's state.
- **CSS**: Styled components for an aesthetically pleasing design.

## Project Structure

The project consists of a single-page application with the following structure:

- **`src/`**: Main source code directory.
  - **`components/`**: Contains React components.
    - **`TestimonialSlider.js`**: Implements the testimonial slider.
    - **`Testimonial.js`**: Represents an individual testimonial.
  - **`assets/`**: Holds images of individuals for testimonials.
  - **`App.js`**: Main application component.
  - **`App.css`**: Stylesheet for the application.

## Usage

## Clone the Repository:
  
   ```bash
   git clone https://github.com/your-username/testimonial-slider.git
   cd testimonial-slider
   ```
## Install Dependencies:
  
  ```bash
  npm install
  ```
## Run the Application:

  ```bash
  npm run start
  ```
